# AlgoFlow 

**Master algorithms visually.**

AlgoFlow is an AI-powered coding companion that instantly transforms your code into interactive flowcharts. Designed for students, developers, and visual learners, it bridges the gap between abstract logic and clear understanding.

## What is AlgoFlow?

Writing code is easy; understanding complex logic is hard. AlgoFlow takes your Python or JavaScript code and uses Gemini AI to generate a step-by-step visual representation of how your algorithm runs.
